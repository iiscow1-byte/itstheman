# Integrated Multilist
Multi list integrator, based off of this mod (https://github.com/hiimjasmine00/IntegratedDemonlist) by hiimjasmine00

## Features
- A new button in the level search screen that opens a menu of various level lists
- A search box that allows you to search for any level in each list by name
- Text on a demon's search box that states its position on the list (If on the list and the text is enabled in the mod settings)
- Gives levels on the ALL their tier
- Gives unrated levels on the ALL the demon icon that roughly corresponds to their difficulty
